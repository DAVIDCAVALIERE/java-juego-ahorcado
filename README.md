# Juego Ahorcado en Java
Hice un juego de ahorcado en java, donde por consola se va introduciendo letra por letra con cierta cantidad de intentos. Si el usuario logra completar toda la palabra secreta gana, caso contrario pierde.

# ¿Como se realizo?
Se uso:
1.  Clase Scanner
2.  Declaraciones y asignaciones
3.  Arreglos
4.  Estructura de control: Iterativa (Bucle)
5.  Clase String y sus métodos valueOf, charAt y length()
6.  Clase Character y su método toLowerCase
7.  Estructura de control: condicional
